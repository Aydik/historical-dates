## Установка и запуск

### 1. Клонирование репозитория

```bash
git clone https://github.com/aydik/historical-dates.git
```

### 2. Установка зависимостей

```bash
npm install
```

### 3. Запуск в режиме разработки

```bash
npm run dev
```

Приложение будет доступно по адресу: http://localhost:3000
